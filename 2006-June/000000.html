At the iSummit later this month, Creative Commons will be shipping
ccPublisher 2.2.  ccPublisher 2.2 will add internationalization support
to ccPublisher, allowing it to be translated into multiple languages and
providing complete support for Unicode character sets.  Tomorrow we will
be releasing a beta of ccPublisher 2.2 as a preview for translators.
This release will incorporate all the bug fixes found in ccPublisher
2.0.4 (also to be released tomorrow), along with the new translation
support.  At this time we know that some bugs probably still exist.  We
hope to track down many of them, as well as give translators a head
start with this beta release.

We'll send another email with release information when the packages are
available.  In the interim, if you are interested in participating as a
translator, you can do the following:

* Subscribe to the cctools-i18n list at Berlios.de; this list will be
used to coordinate translation efforts and answer questions.  If you
have questions about how translation works, policies, etc, this is the
place to ask them.  Subscribe online at
http://lists.berlios.de/mailman/listinfo/cctools-i18n
* Read through our translation instructions at
http://wiki.creativecommons.org/Translating_ccPublisher

We currently only have instructions up for translating on Linux; we'll
be finishing Windows and Mac OS X instructions shortly.

Thanks,
