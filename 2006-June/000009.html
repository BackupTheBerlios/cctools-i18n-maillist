Several of you reported problems with the previous beta release (2.1.0),
in particular crashes when you tried to run ccPublisher with your new
translation.  ccPublisher 2.1.1 (2.2 beta 2) is now available.
Downloads links and release notes are at
http://wiki.creativecommons.org/CcPublisher_2_2.  Windows and Linux
builds are available now; Mac OS X builds will be available in a few hours.

In addition to several bug fixes, the translation catalog templates
included with this release have been extensively updated since the
previous release. These updates include strings which were missed, as
well as additional strings introduced during bug fixes (error messages,
etc). I'll be adding instructions to the Translating ccPublisher page
(http://wiki.creativecommons.org/Translating_ccPublisher) shortly with
information on using msgmerge to merge the new catalogs with the work
you've already completed.  My apologies for any wasted work; the new
catalogs are far more complete and this should be the last rather
seismic shift in catalog content.

The "How Does This Work" window is still untranslated; instructions for
translating that window should be available shortly.

As always, feedback, suggestions and bug reports are critical to making
this a successful release.  Questions regarding translating the app can
be directed to the list; bug reports, etc can be recorded in the Tracker
(http://roundup.creativecommons.org/ccpublisher).  Just a reminder that
translations must be in by 1600 UTC on 20 June 2006 for inclusion in the
ccPublisher 2.2 release for the iSummit.

Thanks,

Nathan
