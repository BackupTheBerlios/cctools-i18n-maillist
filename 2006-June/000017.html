I'd like to thank everyone for all the bug reports and hard work with
translations over the past week while I've been away on holiday.  I
wanted to let everyone know where we are with the ccPublisher 2.2
release, in particular internationalization.

Several of you have pointed out that there were a few remaining strings
missing from the catalog templates, or that strings in the catalog
weren't being rendered properly.  Both of those issues have been
corrected.  I have attached the updated .pot files to this message;
they're also available from Subversion (or through the web at
http://svn.berlios.de/viewcvs/cctools/publisher/trunk/resources/locale/).

An issue with right-to-left language rendering was also reported.
Unfortunately we've found that ccPublisher's underlying toolkit,
wxWidgets, does not yet properly support RTL rendering.  We're exploring
ways to handle RTL, but RTL language support will not be in 2.2.

I'm currently working on some network communication issues for
ccPublisher 2.2 (catching exceptions, etc).  We'll be pushing out an
updated beta release later this evening which will include the bug
fixes, updated catalogs and any translation files I've received to this
point.  As a reminder, a complete translation includes two .po files
(ccpublisher.po and p6.po) and one .html file (welcome.html).
Translations received by 1600 UTC tomorrow (20 June 2006) will be
included in the initial 2.2 release.  Those received after that point
will be rolled into the first bug fix release.

Thanks again to everyone for their hard work.

Nathan
