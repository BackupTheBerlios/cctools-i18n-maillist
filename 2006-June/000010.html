Dear Nathan and all,

There is a string that states "Create metadata suitable for use with 
files hosted on your web site".

I think this might be confusing to a casual user, for whom the 
distinction between hosting files on a server - displaying / making them 
available on a web site might not be clear. I think this string should 
read "on your own server" or something similar. Because one can host 
files at the Archive, but offer access to them not through the archive 
page, but own page, and feel they are hosted on his web site.


Second issue I have - there is a string "ccPublisher" - do we translate 
that? The tool can for instance be called in Polish ccWydawca, but I 
think that would be confusing and in general software names don't seem 
to be translated. Am I right in leaving it as it is?

best,
