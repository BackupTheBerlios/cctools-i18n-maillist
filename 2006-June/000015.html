As some of you have pointed out, there are no strings in the translation
catalogs for the "How Does This Work" dialog in ccPublisher.  This is
due to the fact that the dialog is actually a bit of HTML instead of
typical WX widgets like the rest of the application.

I've now updated the code base to allow translation of the "How Does
This Work" dialog.  In order to translate this window, a copy of the
HTML file must be translated.  The file, welcome.html, is located in the
same directory as the .pot message catalog templates.  You can find a
copy in Subversion at
http://svn.berlios.de/viewcvs/cctools/publisher/trunk/resources/locale/welcome.html;
 just click the "download" or "as text" link to get a copy of the file.
 When submitting your translation, include a translated copy of
welcome.html with your .po files.  If a translated copy of welcome.html
is not available, ccPublisher will fall back to English.

Please let me know if you have any questions about welcome.html or about
translating ccPublisher.
