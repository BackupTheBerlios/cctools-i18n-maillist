A new beta of ccPublisher 2.2 is now available.  This release, 2.1.3,
corrects:

* several critical bugs which caused 2.1.2 to crash when submitting to
archive.org
* display bugs which caused some translated strings to still show in English
* additional internal translation bug fixes

This release includes the following translations:
    * Catalan (ca_ES)
    * Chinese - Taiwan (zh_TW)
    * Croation (hr_HR)
    * English (en_US)
    * Polish (pl_PL)
    * Spanish (es_ES)

Additional translations received by 1600 UTC are guaranteed to be rolled
into 2.2.  Those received after that point will be included if possible,
or will be rolled into the next bug fix release (2.2.1).  This release
is considered feature complete for the 2.2 release.  If additional bugs
or crashes are reported (which of course is possible), every effort will
be made to correct them before the final 2.2 release.

Download links available at
http://wiki.creativecommons.org/CcPublisher_2_2.  Linux and Windows
builds are currently available.  Mac OS X builds will be available later
today.
