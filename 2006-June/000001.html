Pre-release (beta) builds of ccPublisher 2.2 are now available for
downloading and translating.  Download information and basic release
notes are available at http://wiki.creativecommons.org/CcPublisher_2_2. 
Information on translating ccPublisher (including instructions for
Windows and Mac OS X users) is available at
http://wiki.creativecommons.org/Translating_ccPublisher. 

As stated in yesterday's pre-release announcement, ccPublisher 2.2
focuses on internationalization and will be released at the iSummit
later this month.  This release is intended for users who wish to test
the application and provide non-English translations.  Users interested
in helping with translations should join the cctools-i18n list
(http://lists.berlios.de/mailman/listinfo/cctools-i18n).
