Hi everybody!

I finally got some time to translate the final few strings of 
ccPublisher in Dutch. So ccPublisher is now available in Dutch as well! 
I'll ask some fellow Dutch people to check my translation for misstakes.

During the translating I noticed some strange things:

1) Quite a few strings are being used multiple times, so I had to 
translate them multiple times. Is this a feature or a bug ;) ?

2) Maybe its just me, but some of the English text is kinda cryptic or 
'doesn't feel right'. Such as:

- "That identifier does not conform to the Internet Archive's naming 
standards."

Shouldn't that be changed to this?

- "That identifier is not available."

Same here.

- "The interfaces which metadata applies to."

I have no clue what this means....

3) I discovered a small typo:

- "ccPublisher has assembled a suggested identifier based on the 
metadata you provided. Identifiers must be at least five characters long 
and may contains(...)"

Typo: contains should be contain


BTW, how can the average ccPublisher user download a Dutch version of 
ccPublisher?

I mean, I can swap the EN .PO files for the NL files, but I think that 
this might be too much to ask from an average user. Are we going to 
include all the *.PO files and create a multilingual version or are 
there other ideas on how to solve this?
