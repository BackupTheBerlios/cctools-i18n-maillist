Mike Linksvayer wrote:
> On Mon, 2006-07-24 at 17:22 +0200, Bjorn Wijers wrote:
>> BTW, how can the average ccPublisher user download a Dutch version of 
>> ccPublisher?
>>
> 
> I believe the average user will have to wait for a point release that
> includes new translations.  
> 

Unfortunately that is currently the case.  We're trying to do a major
release (i.e., 2.0, 2.2, etc) every 2-4 months and bug fix releases as
needed to minimize the lag, but it does exist.

There are a couple of ideas we're looking at that will help eliminate
this -- one is downloaded "language packs" for the application
(downloaded by ccPublisher itself so no user intervention required, only
permission).

The other is getting the process of "releasing" down to a science so
that it's trivial to make a release.  Part of this would involve
starting to do nightly builds where an automated system checked out the
code, ran the tests and built packages, hopefully on multiple platforms.
 Once that was in place, providing "unofficial" builds with the updated
translations would be trivial, and new "official" releases on slightly
less so.

So those are the two ideas we're kicking around, and we may end up doing
both.  Unfortunately I don't have any sort of time line for either.  If
you have feedback or suggestions, I'd love to hear them.
